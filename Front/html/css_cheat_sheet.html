Margin & Padding

The margin and padding propierties is used to set the spaces of an element.

Overflow, Display & Position

The overflow propierty is used to set the effect of placing one element on top of another element.
The display propierty is used to set whether an element is treated as a block-level element, an inline-level element, or a flex container.
The position propierty is used to set the position of an element relative to its normal position in the document.
